### 75. Работа с деревом узлов

Дан код:
```csharp
public class Node { 
    public Node Parent {get;set;} 
    public string Name {get;set;} 
    public IEnumerable<Node> Children {get;set;} 
}
```

У корневого узла Parent всегда равен null.

Предоставлен корневой узел дерева, содержащий несколько уровней:
```csharp
Node root = ReadXml();
```

**Задания:**
1. Напишите Extension-метод, возвращающий Уровень текущего узла относительно корня
2. Напишите метод рекурсивного обхода дерева, выводящий на консоль узлы в формате Уровень-Наименование
